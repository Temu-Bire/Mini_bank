# 🏦 Mini Bank CLI

A simple command-line banking system built in Python. It allows users to create accounts, deposit, withdraw, transfer funds, check balances, view transaction history, calculate interest, and delete accounts — all with data persisted in a JSON file.

Perfect for learning Python, Pydantic validation, error handling, and basic banking logic!

## ✨ Features

- Create new bank accounts with owner name and initial deposit
- Deposit and withdraw money
- Check account balance
- Transfer funds between accounts
- View transaction history
- Calculate simple interest on an account
- Delete accounts (only if balance is zero)
- Input validation using **Pydantic**
- Custom exceptions and comprehensive error handling
- Persistent storage using JSON
- Logging for all operations

## 📋 Requirements

- Python 3.8 or higher
- `pydantic` library
